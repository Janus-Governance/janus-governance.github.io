# Janus Public Validation Milestone — March 2026

In March 2026 the Janus Governance project reached its first fully
public validation milestone.

The public architecture was published at:

https://janusgovernance.org

The site exposes the following project layers:

- Core
- Documentation
- RFC
- Runtimes
- Demos

The Demos section presents the deterministic validation layer of
Janus Core Lite.

The validation suite is implemented in the repository:

https://github.com/Janus-Governance/janus-demo

The suite generates a public artifact:

janus-validation-suite.json

This artifact is published and consumed by the public website to
display the current governance validation status.

This milestone represents the first complete public cycle of the
Janus architecture:

model → implementation → validation → public verification.
